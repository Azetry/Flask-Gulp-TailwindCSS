Flask Gulp TailwindCSS Framework
================================
# Description
# Python Package need to install
+ Flask
# Usage
+ copy config file and edit it
```cp config.py.example config.py```
+ install frontend
    + ```cd static```
    + ```yarn install```
+ compile
```yarn run dev```
+ build
```yarn run prod```
# Run
```python app.py```
